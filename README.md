## 📝 How to Run Locally

1. **Clone the repo:**
   ```bash
   git clone https://github.com/ruturajsolanki/Portfolio.git
   cd Portfolio
   ```

2. **Frontend:**
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. **Backend:**
   ```bash
   cd ../server
   npm install
   npm run dev
   ```

4. **Environment Variables:**  
   Create a `.env` file in `server/` with your MongoDB and email credentials.
